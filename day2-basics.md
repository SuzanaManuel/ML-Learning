# Day 2 – Basics of Machine Learning 🧠

## 📊 What is Data?
Data is the foundation of Machine Learning. Models learn from data.

Types of data:
- Structured data (tables, Excel)
- Unstructured data (images, text)

---

## ⚙️ What is a Model?
A model is a system that learns patterns from data and makes predictions.

---

## 🔄 Training vs Testing

- Training Data → used to train the model  
- Testing Data → used to check performance  

---

## 💡 What I Learned Today
- Data is more important than algorithms  
- Models improve with better data  
- ML is about patterns, not just coding
